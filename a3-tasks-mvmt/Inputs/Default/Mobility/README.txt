********************************************************************IMPORTANT********************************************************************

- "UserMovementsListEventsDefault.txt" contains a list of all users events, in the default environment. An event is a new user position 
                                       inside the city. Each event is characterized by:

	- ID-User: user identifier.
	- Lat, Long, Alt: geographic location of each event.
	- Day: day of simulation.
	- Hour, Minute: time identifier of an event.

All information are regarding Luxembourg city.
